Do you want to deliver true Digital Experiences? These days that means mastering a range of disciplines including backend and bleeding edge frontend technologies, while targeting a broad number of audiences using different platforms and devices. You should also have security, scalability and performance in mind, while keeping a superb interactive user experience. The Plone Conference is the place where you can truly live tutorials, talks and sprint about Plone CMS stack, Python Web, and Frontend Technologies in an outstanding Open Source community atmosphere. Even if you are not a developer you can take advantage an specific Content and Assets Management track.
